Key UI Decisions

Classic Mode = Indigo
Active pill background uses indigo/blue tones to match the existing info panel styling
Ghostty Mode = Purple
Purple distinguishes it from Classic and matches the scripts button color family
No Input Bar in Ghostty
The ghostty canvas fills from header to bottom. Input goes directly to the PTY via the canvas. No separate input element needed.
Shared Header
Both modes share the same header component. Only the content area below changes. Scripts, copy, and info buttons appear in both modes.
